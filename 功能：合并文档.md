# 报文合并工具 - 需求文档

## 1. 功能概述

将ADS-B报文文件和雷达数据报文文件按时间戳顺序合并，生成统一的A3H格式输出文件。

## 2. 输入格式

### 2.1 ADS-B 报文文件

文件包含两种报文类型：`$AP`（位置报文）和`$AV`（速度报文）

**$AP 位置报文格式：**
```
$AP,icao_hex,yyyy-mm-dd-HH-MM-SS,ms,ns,lon,lat,altitude,quality,...
```
示例：
```
$AP,500408,2026-02-04-14-29-00,0,0,132.23,32.4,13106.400,6,213453.451,0.00,2.021
```

**$AV 速度报文格式：**
```
$AV,icao_hex,yyyy-mm-dd-HH-MM-SS,ms,ns,speed,heading,vertical_speed,...
```
示例：
```
$AV,500408,2026-02-04-14-29-00,0,0,399,240,0,0,0
```

**ADS-B时间格式：**
- 主时间：`yyyy-mm-dd-HH-MM-SS`（年-月-日-时-分-秒）
- 毫秒字段：`ms`（0-999）
- 纳秒字段：`ns`（0-999999）

### 2.2 雷达报文文件

文件包含`$RD`（雷达数据报文）

**$RD 雷达报文格式：**
```
$RD,track_id,ms_from_midnight,0,2,0,0,0,0,0,lon,lat,0,0,range,az,0,0,0,0,0
```
示例：
```
$RD,7082,62011012,0,2,0,0,0,0,0,135.78220729,32.25913567,0,0,4197.82802208,89.43500280,0,0,0,0,0
```

**雷达时间格式：**
- 毫秒数：从当天00:00:00开始的毫秒数（0-86400000）

## 3. 输出格式

合并后的文件包含三种报文类型：`$AP`、`$AV`、`$RD`，按时间戳升序排列。

输出文件采用与雷达报文相同的 `$RD` 格式，或者保留原始报文格式按时间排序输出。

**统一时间戳转换规则：**
- ADS-B时间 → 转换为毫秒数：`HH*3600000 + MM*60000 + SS*1000 + ms`
- 雷达时间 → 直接使用毫秒数

## 4. 处理逻辑

### 4.1 时间戳解析
1. 读取ADS-B报文，解析 `yyyy-mm-dd-HH-MM-SS` + `ms` → 统一转换为毫秒
2. 读取雷达报文，直接使用毫秒数字段

### 4.2 排序合并
1. 解析所有报文的时间戳
2. 按毫秒数升序排序
3. 输出到统一的合并文件

### 4.3 约束条件
- 输入文件支持GBK/UTF-8编码
- 支持命令行参数指定输入输出文件
- 支持指定参考日期（用于ADS-B时间戳转换）

## 5. 命令行接口

```bash
python merge_messages.py <adsb_file> <radar_file> <output_file> [options]

参数：
  adsb_file      ADS-B报文文件路径
  radar_file     雷达报文文件路径
  output_file    合并输出文件路径

选项：
  --date DATE    参考日期 (格式: yyyy-mm-dd)，用于转换ADS-B时间戳
  --encoding ENC 输入文件编码 (默认: gbk)
  -v, --verbose  显示详细输出
```

## 6. 使用示例

```bash
# 基本用法
python merge_messages.py data/sample/2026-02-04.a3h data/sample/track_20251210172235.a3h data/sample/merged.a3h

# 指定参考日期
python merge_messages.py adsb.a3h radar.a3h merged.a3h --date 2026-02-04
```

## 7. 数据样例

### 输入1: ADS-B报文 (2026-02-04.a3h)
```
$AP,500408,2026-02-04-14-29-00,0,0,132.23,32.4,13106.400,6,213453.451,0.00,2.021
$AV,500408,2026-02-04-14-29-00,0,0,399,240,0,0,0
$AP,789244,2026-02-04-14-29-01,1,1000001,128.07,29.6,12496.800,6,213453.451,0.00,2.021
$AV,789244,2026-02-04-14-29-01,1,1000001,596,51,0,0,0
```

### 输入2: 雷达报文 (track_20251210172235.a3h)
```
$RD,7082,62011012,0,2,0,0,0,0,0,135.78220729,32.25913567,0,0,4197.82802208,89.43500280,0,0,0,0,0
$RD,7175,62011014,0,2,0,0,0,0,0,128.91492422,29.84437293,0,0,3094.41375741,103.71290224,0,0,0,0,0
```

### 输出: 合并后报文
```
$RD,7082,62011012,...
$RD,7175,62011014,...
$AP,500408,2026-02-04-14-29-00,...
$AV,500408,2026-02-04-14-29-00,...
$AP,789244,2026-02-04-14-29-01,...
$AV,789244,2026-02-04-14-29-01,...
```
(按时间戳升序排列)

## 8. 字段说明

| 字段 | 说明 |
|------|------|
| icao_hex | ADS-B航班标识（6位十六进制） |
| track_id | 雷达航迹编号 |
| ms_from_midnight | 当天毫秒数 |
| lon | 经度 |
| lat | 纬度 |
| altitude | 高度（米） |
| speed | 地速（节） |
| heading | 航向角（度） |
| range | 距离（米） |
| az | 方位角（度） |
